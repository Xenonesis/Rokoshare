## 🚀 About RokoShare

RokoShare is a modern shared mobility platform designed to transform daily commuting in Gurugram. We provide affordable, reliable, and eco-friendly shared auto rides on fixed routes, making urban transportation accessible to everyone.

### Our Mission
- **Affordability**: Make daily commuting budget-friendly for everyone
- **Reliability**: Provide consistent, on-time service you can depend on
- **Sustainability**: Reduce carbon footprint through smart shared transportation
- **Convenience**: Simplify the daily commute with fixed routes and predictable pricing

### Our Vision
To become the leading shared mobility platform in India, connecting communities through smart, affordable, and eco-friendly transportation solutions.

## ✨ Features

### 🎯 Core Features
- **Fixed Route Network**: 50+ strategically located routes across Gurugram
- **Affordable Pricing**: Save up to 60% compared to traditional auto services
- **Real-Time Tracking**: Live GPS tracking for complete transparency
- **Mobile-First Design**: Responsive design that works seamlessly on all devices
- **24/7 Support**: Round-the-clock customer assistance

### 🎨 User Experience
- **Modern UI/UX**: Clean, intuitive interface with smooth animations
- **Dark Mode Support**: Easy on the eyes for night-time commuters
- **Quick Booking**: One-tap ride booking with minimal steps
- **Payment Integration**: Multiple payment options for convenience
- **Ride History**: Track all your past rides and expenses

### 🛡️ Safety & Security
- **Verified Drivers**: All drivers go through rigorous background checks
- **Emergency Button**: Instant access to support during rides
- **Share Location**: Share your ride details with trusted contacts
- **Safety Ratings**: Transparent driver rating system

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for CDN resources

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Xenonesis/Rokoshare.git
   cd Rokoshare
   ```

2. **Open in browser**
   ```bash
   # Open index.html in your preferred browser
   # Or use a local server:
   # Python 3
   python -m http.server 8000
   # Node.js
   npx http-server
   ```

3. **Explore the application**
   - Navigate through different pages using the menu
   - Test all interactive elements
   - Check responsiveness on different screen sizes
