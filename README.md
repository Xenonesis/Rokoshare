## ğŸš€ About RokoShare

RokoShare is a modern shared mobility platform designed to transform daily commuting in Gurugram. We provide affordable, reliable, and eco-friendly shared auto rides on fixed routes, making urban transportation accessible to everyone.

### Our Mission
- **Affordability**: Make daily commuting budget-friendly for everyone
- **Reliability**: Provide consistent, on-time service you can depend on
- **Sustainability**: Reduce carbon footprint through smart shared transportation
- **Convenience**: Simplify the daily commute with fixed routes and predictable pricing

### Our Vision
To become the leading shared mobility platform in India, connecting communities through smart, affordable, and eco-friendly transportation solutions.

## ğŸš€ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for CDN resources

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Xenonesis/Rokoshare.git
   cd Rokoshare
   ```

2. **Open in browser**
   ```bash
   # Open index.html in your preferred browser
   # Or use a local server:
   # Python 3
   python -m http.server 8000
   # Node.js
   npx http-server
   ```

3. **Explore the application**
   - Navigate through different pages using the menu
   - Test all interactive elements
   - Check responsiveness on different screen sizes
